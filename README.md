# dffp
Various tooling, proof-of-concepts, and AWS IAC for DFFP in Clash.

# Prerequisites (macOS)
AWS CLI
```bash
brew install awscli
```
Terraform
```bash
brew tap hashicorp/tap
brew install hashicorp/tap/terraform
terraform -install-autocomplete
```
Pyenv
```bash
brew install pyenv
```
Node Version Manager
```bash
brew install nvm
nvm install 20.12.2
nvm use 20.12.2
```
pnpm
```bash
npm i -g pnpm@9.9.0
pnpm install
```
